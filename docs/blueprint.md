# **App Name**: BachhaBite

## Core Features:

- User Authentication & Profiles: Customers can register and login using their mobile number or email. Dedicated login panels for Admin and Delivery Boys.
- Daily Menu & Subscription Display: Customers can browse daily menu items with details like name, type, time slot, price, and description. Displays monthly subscription options.
- Order Placement & Management: Customers can place daily orders or subscribe monthly for selected items, customizing quantity, type, and delivery time. Includes order history and payment status.
- Admin Menu & Order Oversight: Admin can add, edit, or delete daily menu items, view all incoming orders (daily and subscriptions), and filter them by various criteria.
- Delivery Assignment & Tracking: Admin can assign orders to Delivery Boys. Delivery Boys can log in, view their assigned orders with customer details, and update order status (Out for Delivery, Delivered).
- Customer Dashboard & Notifications: Customers have a dashboard showing today’s order status, upcoming subscription orders. Receives notifications/messages when Admin posts daily menus.
- AI-powered Menu Notification Tool: An internal tool for the Admin to generate and broadcast engaging, personalized push notifications about new daily menus or subscription updates to customers using generative AI.

## Style Guidelines:

- Primary color: A warm coral (#EB7E47) to evoke an inviting and appetizing feel, reflecting the food brand and energy. Calculated from HSL(20, 80%, 60%).
- Background color: A very soft, desaturated peach (#FCF5F3) to provide a light, airy, and clean canvas. Calculated from HSL(20, 20%, 95%).
- Accent color: A vibrant deep red-pink (#C7264B) for calls to action, important highlights, and interactive elements, providing strong contrast. Calculated from HSL(350, 70%, 50%).
- Body and headline font: 'PT Sans' (humanist sans-serif) for its modern yet friendly aesthetic, ensuring excellent readability across all sections and device sizes, fitting the 'kids/food brand' theme.
- Use clear, simple, and filled icons that are easily recognizable within card-based layouts and order timelines. Icons should intuitively represent menu categories, order status, and navigational elements for quick comprehension.
- Implement a modern, responsive, mobile-first design strategy. Menu items and order information should be presented in digestible card layouts, and order tracking should utilize a clear timeline-style visualization for progress.
- Incorporate subtle and smooth animations for user feedback on interactive elements, state changes (e.g., order updates), and seamless transitions between different views, enhancing the overall user experience.
